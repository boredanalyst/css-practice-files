Box Model

Each element in the document is placed in a box.

Displays

Any element can be either inline, block, or inline-block.

Some elements are naturally inline or block, but you can edit this property through CSS.

Block Properties

Block properties are always situated alone in one line.

They also span the entire line.

I am a DIV and I am a block element.

Inline Properties

Inline elements are situated in the same line as other elements.

Their width is dictated by the width of their content.
