Introduction to Selectors

Selectors are a way for you to specifically target elements in your webpage and style them accordingly.

Element Selectors

The second item in this list is colored pink due to the tag selector.

The tag or element selector affects all specified elements.

However, because the first and last items are affected by class and id selectors respectively, those take precedence.

  1. I am the first item with a class of 'red'
  2. I am the second item with no class or id.
  3. I am the third item with the id 'myListItem'
